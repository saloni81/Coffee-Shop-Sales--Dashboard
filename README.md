# ☕ Coffee Shop Sales Dashboard (Excel)

An interactive **Coffee Shop Sales Dashboard** built in **Microsoft Excel** to analyze sales performance, customer footfall, product demand, and store-wise performance using Pivot Tables, Pivot Charts, Slicers, and KPI cards.

---

## 📌 Project Overview

This dashboard provides valuable business insights by transforming raw coffee shop sales data into an interactive visual report. Users can filter data by **Month** and **Day** to analyze sales trends, customer behavior, and product performance.

## 📊 Dashboard Insights

### Sales Performance
- Displays total sales generated across all stores.
- Calculates average bill value and customer orders.

### Customer Footfall
- Tracks total customer visits.
- Compares footfall across different store locations.

### Hourly Analysis
- Shows quantity ordered by hour.
- Identifies peak business hours.

### Category Analysis
- Percentage contribution of each product category.
- Helps identify highest revenue-generating categories.

### Drink Size Analysis
- Order distribution among:
  - Large
  - Regular
  - Small
  - Not Defined

### Store Performance
Comparison between:
- Astoria
- Hell's Kitchen
- Lower Manhattan

### Product Performance
Top 5 best-selling products based on sales.

### Weekday Analysis
Compares orders across all days of the week.
